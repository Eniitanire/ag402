ag402 — live payment flow
1. Agent requests GET /api/token-audit
2. Server responds 402 Payment Required — $0.05 USDC
3. Ag402 checks budget: $0.05 ≤ $5.00 limit ✓
4. Solana on-chain transfer: 0.05 USDC → seller
5. Retry with proof: Authorization: x402 ...
6. Server responds 200 OK — audit report received
✓ Transparent to your agent — it only sees the 200
